# AI Contribution Policy

AgentVoir allows AI-assisted code contributions, but all AI-generated changes must be reviewed by a human maintainer before merge.

## Rules

1. AI bots may open issues, branches, comments, and pull requests.
2. AI bots may not merge to `main`.
3. AI bots may not approve their own pull requests.
4. AI-generated pull requests must be labeled `ai-generated`.
5. AI-generated pull requests must disclose the tool or bot used.
6. Human maintainers are responsible for correctness, security, licensing, and maintainability.
7. No secrets, credentials, private data, or proprietary code may be included in AI-generated changes.
8. Security-sensitive changes require explicit maintainer review.
